## 🔧 Personnalisation

### Changer le Numéro
Cherchez et remplacez dans `index.html` :
- `+33123456789` → Votre numéro international
- `01 23 45 67 89` → Votre numéro formaté

### Changer les Couleurs
Dans `src/style.css` :
```css
:root {
  --accent-red: #E94560;  /* Couleur principale */
  --primary-blue: #0F3460; /* Bleu foncé */
}
```
